Library of Classical Books

База данных мировой классической литературы
Разработчик: Чухланцев Данил Алексеевич



О проекте

Проект представляет собой базу данных для библиотеки классической литературы. База содержит информацию о книгах, авторах, литературных направлениях, читателях и выдаче книг.



Структура базы данных

База данных состоит из 8 таблиц:

Таблица                      Описание
---------------------------  -----------------------------------------
countries                    Страны происхождения книг и авторов
literary_movements           Литературные направления
genres                       Жанры книг
authors                      Информация об авторах
books                        Информация о книгах
editions                     Издания книг
readers                      Читатели библиотеки
loans                        Выдача и возврат книг



Технологии

- PostgreSQL (система управления базами данных)
- Python (язык программирования)
- psycopg2 (библиотека для подключения к PostgreSQL)
- Git (система контроля версий)
