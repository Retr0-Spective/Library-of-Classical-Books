LIBRARY OF CLASSICAL BOOKS

База данных мировой классической литературы

Разработчик: Чухланцев Данил Алексеевич Б24-191-1зу



О ПРОЕКТЕ

Проект представляет собой базу данных для библиотеки классической литературы.
База содержит информацию о книгах, авторах, литературных направлениях, читателях и выдаче книг.



СТРУКТУРА БАЗЫ ДАННЫХ

База данных состоит из 8 таблиц:

|Таблица|Описание|
|-|-|
|countries|Страны происхождения книг и авторов|
|literary\_movements|Литературные направления|
|genres|Жанры книг|
|authors|Информация об авторах|
|books|Информация о книгах|
|editions|Издания книг|
|readers|Читатели библиотеки|
|loans|Выдача и возврат книг|



ТЕХНОЛОГИИ

* PostgreSQL (система управления базами данных)
* Python (язык программирования)
* psycopg2 (библиотека для подключения к PostgreSQL)
* Git (система контроля версий)



ЗАПУСК ПРОЕКТА

1\. Установите PostgreSQL и создайте базу данных:

&#x20;  CREATE DATABASE classic\_library ENCODING 'UTF8'(Обязательно в кодировке UTF-8 так как русские символы в иной кодировке допустим в кодировке по умолчанию(WIN1251) будут отображаться как не понятные каракули);



2\. Выполните SQL-скрипты:

&#x20;  psql -U postgres -d classic\_library -f create\_tables.sql

&#x20;  psql -U postgres -d classic\_library -f insert\_data.sql



3\. Установите Python библиотеку:

&#x20;  pip install psycopg2



4\. Запустите приложение:

&#x20;  python library\_app.py
